1.2 Com funciona la web A la web tenim: Els servidors Hardware o màquina virtual amb domini donat d’alta, IP pública i programari instal·lat tipus apache, tomcat, jboss o IIS entre altres. Els navegadors Chrome, Firefox, Safari, Internet Explorer finalment tots compleixen uns estàndards, però ara tenim el problema de diferents dispositius amb diferents tipus d'accés (touchpad o teclat i ratolí) i diferent resolució de pantalla. Les url El maneig de les URL és una cosa quotidiana quan ens movem per Internet, ja que un percentatge molt alt del temps ens el dediquem a escriure al navegador adreces URL. L'estructura general d'una URL és la següent: protocol://servidor:port/directori/pàgina Protocol, és el format en què es van a intercanviar les dades nostre navegador i la màquina de destinació: HTTP, HTTPS, FTP,... Servidor, és el nom de la màquina destinació (DNS). Nota: pot haver subdominis que poden representar noms de màquines. Port: HTTP té el port 80 (més a Xarxes) Ok, ja ho tenim tot, anem a veure com funciona tot junt.
